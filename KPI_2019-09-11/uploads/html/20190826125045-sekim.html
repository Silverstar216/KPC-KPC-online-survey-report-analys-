주의하십시오.

1.  redhat 7.0 에 맞추었습니다.

2.  다국어를 지원하도록 모든 언어를 선택하십시오.

한국어나 중국어가 없으면 문서변환시 폰트가 깨집니다.

3.  devtools를 모두 선택해서 설치하셔야 프로그램이 자동으로 설치됩니다.

설치방법

==================================================================

sudo chmod 775 tohtmlEX_install.sh

sudo ./tohtmlEX_install.sh

Machine Number.txt를 sgchoo77@hanmail.net로 보내주십시오.

Serial Number를 보내드리면

/opt/hwp2htmlEX 디렉토리에 복사하십시오.

파일변환 방법

==================================================================

1. hwp파일을 html파일로 변환하는 코맨드

cd /opt/hwp2htmlEX

./conv -s "/tmp/001.hwp" -o "/tmp/001.html" -m convert

/tmp/001.html파일이 생성됩니다.

2. pdf를 html파일로 변환하는 코맨드

pdf2htmlEX --fallback 1 --process-outline 0 --dest-dir "/tmp" "/tmp/002.pdf"

/tmp/002.html파일이 생성됩니다.

3. office파일을 pdf로 변환하는 코맨드

/opt/libreoffice5.0/program/python /opt/libreoffice5.0/program/conv.py -s "/tmp/003.docx"

/tmp/003.pdf 가 생성됩니다.

pdf2htmlEX --fallback 1 --process-outline 0 --dest-dir "/tmp" "/tmp/003.pdf"

/tmp/003.html파일이 생성됩니다.
