LIT-Generator Client Sample Guide

1．모듈 구성

-　아래 그림과 같이 3개의 JAVA Class와 1개의 jsp 파일로 구성 되어 있습니다.

2. 모듈 설명

- LitCommunicator.java : 서버 통신 클래스

- LitConvert.java : 컨버트 메소드 공통 클래스

- LitDecode.java : VO 클래스

- LitGeneratorClient.jsp : 컨버트 클래스들의 사용 예제

3. 프로세스 설명

4. 샘플 사용

- 첫째, 3개의 클래스를 프로젝트의 사용하고자 하는 위치에 추가합니다.

- 둘째, LitGeneratorClient.jsp 를 테스트 하고자 하는 위치에 추가한 후 브라우저에서 테스트 합니다. (샘플은 컨버트 프로세스의 이해를 돕고자 만들어져 있습니다.)

- 셋째, LitGeneratorClient.jsp 파일을 열어서 필요한 코드를 추가 하여 테스트 합니다.

(특히 컨버트 대상 테스트 파일의 위치를 적절한 위치에 복사한 후 .jsp 파일 내부에 path 값을 정확히 입력해야 합니다.)

- 넷째, 실제 사용하시고자 하는 소스에 LitGeneratorClient.jsp 파일을 참고하여 코드를 작성합니다.

5. 중요 사항

★ Lit-Generator 서비스에서 제공하는 File ID 값을 사용하길 원하지 않으시고, 자체적인 File ID값을 사용하길 원하시는 경우 LitGeneratorClient.jsp 소스 내부에 있는 다음 그림과 같이 LitConvert.isLocal 값을 true 로 설정하여 사용하여 주시기 바랍니다.
